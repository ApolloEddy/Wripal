## [0.1.0] - 2024-12-12

### 🎉 首个开发版本

#### 新增 (Added)
- **书架系统**
  - 书籍创建、编辑、删除
  - 章节管理与排序
  - 全书大纲编辑器
  - 字数统计功能

- **富文本编辑器**
  - 集成 flutter_quill 编辑器
  - 支持标题、粗体、斜体、下划线等格式
  - 支持有序/无序列表、引用
  - 自动保存功能（3秒防抖）

- **响应式布局**
  - 可折叠侧边栏
  - 横屏/竖屏自适应
  - 卡片式模块系统

- **主题系统**
  - 亮色/暗色主题
  - 拟态 UI 风格
  - 圆角设计语言

- **Web 平台支持**
  - 平台感知编辑器 (WebSafeEditor)
  - 使用 GlobalKey 确保内容正确保存
  - 工具栏溢出问题修复

#### 修复 (Fixed)
- 修复 Web 平台编辑器保存失败问题
- 修复 RenderFlex 工具栏溢出错误
- 修复自动保存状态不更新问题

#### 技术栈
- Flutter 3.38.4 / Dart 3.10
- flutter_riverpod 状态管理
- Hive 本地存储
- flutter_quill 富文本编辑
