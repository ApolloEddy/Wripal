# 更新日志 (Changelog)

本文件记录 Wripal 项目的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.2.0] - 2024-12-12

### 🔧 富文本编辑器重构

#### 变更 (Changed)
- **富文本编辑器**: 从 flutter_quill 迁移到 **fleather**
  - 解决 Windows 桌面端兼容性问题
  - 更稳定的编辑体验
  - 保留完整格式功能（加粗、斜体、对齐等）

- **工具栏优化**
  - 工具栏整合到 `ChapterRichEditor` 组件内部
  - 支持隐藏/显示切换
  - 水平滚动避免溢出

- **主题系统**
  - 添加侧栏主题切换按钮
  - 默认主题改为亮色
  - 主题设置持久化到本地

#### 优化 (Improved)
- 简化阴影效果（性能优化）
- 添加 RepaintBoundary 隔离（减少重绘）
- 自动保存防抖改为 2 秒

#### 移除 (Removed)
- 侧栏富文本编辑器卡片
- Google Fonts 网络加载（改用系统字体）
- flutter_quill 依赖

## [0.1.0] - 2024-12-12

### 🎉 首个开发版本

#### 新增 (Added)
- **书架系统**
  - 书籍创建、编辑、删除
  - 章节管理与排序
  - 全书大纲编辑器
  - 字数统计功能

- **富文本编辑器**
  - 支持标题、粗体、斜体、下划线等格式
  - 支持有序/无序列表、引用
  - 自动保存功能

- **响应式布局**
  - 可折叠侧边栏
  - 横屏/竖屏自适应
  - 卡片式模块系统

- **主题系统**
  - 亮色/暗色主题
  - 圆角设计语言

#### 技术栈
- Flutter 3.38.4 / Dart 3.10
- flutter_riverpod 状态管理
- Hive 本地存储
